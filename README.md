## 🧩 How It Works

1. User enters:

   * Name
   * Memory/personality
   * Question

2. Backend:

   * Builds a **system prompt**
   * Sends request to Groq API

3. AI:

   * Generates response in persona

4. Output:

   * Realistic, character-based reply

## ⚙️ Setup Locally

### 1. Clone repository

```id="j0gczw"
git clone https://github.com/akshay212-gif/personaai.git
cd personaai
```

### 2. Install dependencies

```id="kq0u2j"
npm install
```

### 3. Configure environment

Create `.env` file:

```id="r2m4rz"
GROQ_API_KEY=your_api_key_here
```

### 4. Run server

```id="hfjc9c"
node server.js
```
